# 🎁 Advanced Discord Drop Bot

A high-interaction, GUI-driven "drop" bot built with **Discord.js v14**. This bot uses Modals, Buttons, and Role Select Menus to create a seamless giveaway experience.

## ✨ Features
- **Multi-Step Setup**: Use `/drop` to select roles, items, and links via Modals.
- **Math Challenge**: Users must solve a random math problem to claim the prize.
- **Anti-Abuse**: Once a winner is found, the drop is immediately locked.
- **Private Delivery**: Rewards are sent via Direct Messages to the winner.
- **Role Pings**: Automatically pings a selected role to notify users.
